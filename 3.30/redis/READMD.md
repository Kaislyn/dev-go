# Redis 
## 一、Redis 基础介绍
Redis 是**高性能开源内存数据库**，核心是**数据存于内存**，读写速度极快，是 Go 后端常用工具，与 JWT 搭配场景极多。

## 二、核心作用（新手必记）
1. **解决 Session 共享**：多服务器集群时，Session 存入 Redis，全服务器共享，避免用户重新登录。
2. **缓存 JWT 令牌**：快速校验令牌、手动过期（用户登出时删除 Redis 令牌，实现强制登出）。
3. **缓存高频数据**：缓存用户信息、接口结果，减少数据库查询压力，提升响应速度。

## 三、关键特点（新手易懂）
- **快**：内存读写，速度为传统数据库数十~上百倍。
- **轻量**：安装简单、配置简洁，易上手。
- **多功能**：支持字符串、哈希、列表等多种数据类型。
- **可持久化**：数据落盘，避免重启丢失。

## 四、与 Go-JWT 关联
本地测试可暂不用 Redis；**多服务器部署**时，用 Redis 缓存 JWT 令牌/Session，是大厂真实鉴权架构。

## 五、为什么用 Redis（对比 MySQL）
**核心：解决 MySQL 磁盘 IO 性能瓶颈**
- MySQL：数据存硬盘，高并发（秒杀/热搜）易宕机。
- Redis：**内存键值对数据库**，读写速度是硬盘数万倍。

**大厂经典架构**：Redis 前置缓存，请求先查 Redis（命中直接返回），未命中再查 MySQL 并缓存。

## 六、Mac 安装&启动 Redis（Homebrew）
1. 安装：`brew install redis`
2. 后台启动（开机自启）：`brew services start redis`
3. 测试：`redis-cli ping` → 返回 `PONG` 即成功（默认端口 6379）

## 七、Go 操作 Redis（go-redis V9）
Go 生态权威 Redis 客户端，安装命令：
```bash
go get -u github.com/redis/go-redis/v9
```
